# Mbappé Dictator Comic

A comic website celebrating the legendary Kylian Mbappé dictator meme!

## About

This project is a fun, single-page comic website inspired by the viral "Mbappé Dictator" meme that took over the internet. It showcases the energy, speed, and leadership of one of football's greatest players.

## Features

- Responsive design
- Interactive easter egg (click the title!)
- Scroll animations
- Modern dark theme with glowing effects

## Tech Stack

- HTML5
- CSS3 (animations, gradients, flexbox/grid)
- Vanilla JavaScript

## Deployment

This site is deployed on Vercel.
